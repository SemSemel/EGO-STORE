## 🎮 وصف المشروع

**EgoRate Store** هو موقع عربي لشحن الألعاب الإلكترونية بتصميم عصري وتفاعلي. الموقع يدعم حالياً شحن **Free Fire** فقط، مع خطط لإضافة ألعاب أخرى قريباً.

## ✨ المميزات

- 🎨 **تصميم عصري** مع تأثيرات متحركة وخلفيات ديناميكية
- ⚡ **شحن فوري** للجواهر
- 🔒 **100% آمن** مع حماية البيانات
- 💰 **أسعار تنافسية** (100 جوهرة = 280 د.ج فقط!)
- 📱 **متجاوب** مع جميع الأجهزة
- 🎮 **واجهة سهلة** الاستخدام

## 🚀 كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. اختر **Free Fire** (اللعبة الوحيدة المتاحة حالياً)
3. أدخل **معرف اللاعب** (Player ID)
4. اختر **عدد الجواهر**
5. اختر **طريقة الدفع**
6. اضغط **تأكيد الطلب**
